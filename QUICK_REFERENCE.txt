## Common Changes

### Change a button link:
Find the button in HTML and change the `onclick` value or `href` attribute

### Change colors:
Find the hex color code in `css/styles.css` and replace it

### Change text/content:
Edit the text directly in the HTML file

### Add more activities:
Copy an activity card in `activities.html` and modify it

### Update contact info:
Edit the contact details in `contact.html`

## Testing

To test:
1. Open `pages/index.html` in browser
2. Click through all navigation links
3. Test buttons and forms
4. Resize browser to test responsive design
5. Check console for any errors (F12)
