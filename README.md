## File Descriptions

### HTML Pages (pages/ folder)
- **index.html** - Home page with welcome section, about us, and why choose us
- **services.html** - Services offered including equipment rental, training, expeditions, accommodation, dining, and group packages
- **activities.html** - 8 outdoor activities with descriptions, pricing, and booking buttons
- **newsletters.html** - Newsletter subscription form and latest news section
- **contact.html** - Contact information, contact form, and FAQ section

### Styling (css/ folder)
- **styles.css** - Contains all CSS styling for:
  - Header and navigation
  - Hero sections
  - Feature cards and activity cards
  - Form styling
  - Responsive design (mobile, tablet, desktop)
  - Button styles and hover effects
  - Footer styling

### JavaScript (js/ folder)
- **script.js** - JavaScript functionality including:
  - Smooth scrolling for navigation links
  - Page load animations
  - Newsletter form handler
  - Contact form handler
  - Form validation
  - Console welcome message

## How to Use

### To open the website:
1. Navigate to the `pages` folder
2. Open `index.html` in your web browser
3. Use the navigation menu to visit other pages

### File Paths:
- All HTML files link to CSS using: `../css/styles.css`
- All HTML files link to JavaScript using: `../js/script.js`
- All internal navigation links use relative paths: `filename.html`

### To modify styling:
- Edit `css/styles.css` - Changes apply to all pages

### To modify scripts:
- Edit `js/script.js` - Changes apply to all pages

### To modify individual pages:
- Edit the specific HTML file in the `pages` folder
- Content is isolated per page

## Customization Guide

### Change Colors:
Edit the following hex codes in `css/styles.css`:
- `#2c3e50` - Primary dark color
- `#3498db` - Primary blue color
- `#27ae60` - Button green color
- `#f4f4f4` - Background gray

### Change Content:
Edit the specific HTML file to update:
- Text content
- Pricing information
- Contact details
- Links and navigation

### Add New Page:
1. Create new HTML file in `pages/` folder
2. Copy structure from any existing page
3. Update the navigation menu in all pages
4. Add link in header navigation

### Update Navigation:
Edit the `<nav>` section in each HTML file to add or remove links
